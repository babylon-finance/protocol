## Quick Start

Git clone

```bash
git clone https://github.com/babylon-finance/protocol.git
```

Install dependencies

```bash
yarn
```

Compile contracts

```bash
yarn compile
```

Generate mnemonic.txt file

```bash
yarn generate
```
